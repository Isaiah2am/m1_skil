# sushi
![alt text](image.png)
## benodigheden
zee wier
komkommer
rijst
sushi roler
rice vinegar
### bereiding
1 voeg de rice vinegar met de rijst
2 zet het zeewier op de sushi roller
3 zet het rijst op de zewier
4 zet de komkommer in de rijst
5 rol de zeewier
